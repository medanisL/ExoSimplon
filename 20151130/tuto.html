HTML :

Qu'est ce que le HTML??
le html a fait son apparition dès 1991 lors du lancement du Web. Son rôle est de gérer et organiser le contenu. C'est donc en HTML que vous écrirez ce qui doit être affiché sur la page : du texte, des liens, des images… Vous direz par exemple : « Ceci est mon titre, ceci est mon menu, voici le texte principal de la page, voici une image à afficher, etc. ».

Balise

Les pages HTML sont remplies de ce qu'on appelle des balises. Celles-ci sont invisibles à l'écran pour vos visiteurs, mais elles permettent à l'ordinateur de comprendre ce qu'il doit afficher. Les balises se repèrent facilement. Elles sont entourées de « chevrons », c'est-à-dire des symboles <et > , comme ceci <balise> .
Les balises indiquent la nature du texte qu'elles encadrent: on note deux types les balises orphelines (auto-fermante) comme <img/> et les balises en paires ex: <p> </p>

Attribut

Les attributs sont un peu les options des balises. Ils viennent les compléter pour donner des informations supplémentaires et faciliter la communication avec le navigateur. L'attribut se place après le nom de la balise ouvrante et a le plus souvent une valeur.comme ceci: <balise attribut="valeur"/> ; un exemple plus concret <image src="photo.jpg" />

Balises Speciales

Pour faire simple , il exsite des balises generiques : qui ont aucun interet pour le navigateur Contrairement au balises qui ont un sens pour le navigateur .
mais elles peuvent etre tres utile pour l'utilisation du css , on note le <div> qui englobe un bloque de contenu et le <span> qui englobe une plus petite partie de type texte : titre

CSS :

Qu'est ce que le CSS??
Cascading Style Sheets, aussi appelées Feuilles de style : le rôle du CSS est de gérer l'apparence de la page web (agencement, positionnement, décoration, couleurs, taille du texte…). Ce langage est venu compléter le HTML en 1996.

Quelques Régles a respecter pour devenir un bon developpeur ! :

BOX MODEL : standard de dimension d'un bloc avec box-sizing

La propriété CSS box-sizing est utilisé pour modifier le modèle de boîte CSS par défaut qui est utilisé pour calculer la largeur et la hauteur des éléments. Il est possible d'utiliser cette propriété pour émuler le comportement des navigateurs qui n'appliquent pas correctement la spécification du modèle de boîte CSS.

SOC : methode de travail efficace

Approche pour concevoir une solution, qui divise le problème en morceaux pouvant être résolus indépendamment.En developpement Web on separera le html le css et le javascript!

DRY:Don't Repeat Yourself

Ne vous répétez pas est une philosophie en programmation consistant à éviter la redondance de code au travers de l’ensemble d’une application afin de faciliter la maintenance, le test, le débogage et les évolutions de cette dernière.
